# Diseño y Estilo

## Paleta de colores (dark-first)

- **Fondo**: `#0b0b0f`
- **Superficies**: `#11131a` / `#151827`
- **Texto**: `#f2f2f7`

## Espaciado

Valores cuadrados estándar (px): 4, 8, 12, 16, 24.

## Radios

- **Tarjetas**: `rounded-2xl`.
- **Inputs / controles**: `rounded-md`.

## Sombras

- `shadow-sm` / `shadow-md` sutiles para elevar elementos.

## Tipografía

- Tamaños: `base`, `sm`, `lg`, `xl`, `2xl`.
- Tracking compacto en títulos para sensación cinematográfica.

> Estos tokens deben usarse consistentemente en la UI y pueden representarse como variables CSS, tema de Tailwind o similar.
