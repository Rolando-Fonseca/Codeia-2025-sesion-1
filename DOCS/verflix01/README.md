# 🎬 Verflix - MVP

Proyecto de aprendizaje para visualizar películas en tiempo real usando la API de TMDB.

## 🚀 Estado del Proyecto

Actualmente en fase de **Arquitectura y Conexión de Datos**.

## 🛠️ Tecnologías Usadas

- **React.js**: Frontend.
- **TMDB API**: Fuente de datos.
- **Git/GitHub**: Control de versiones.

## 📂 Documentación

Puedes encontrar los detalles del diseño y la planificación en la carpeta `/DOCS`:

- [Modelo de Datos](./DOCS/data_model.md)
- [Plan de Commits](./DOCS/plan_commits.md)
- [Dependencias](./DOCS/dependencias.md)

## ⚙️ Configuración Local

1. Clona el repositorio.
2. Crea un archivo `.env` con tu `REACT_APP_TMDB_API_KEY`.
3. Ejecuta `npm install` y luego `npm start`.
